# 电商销售数据分析系统

基于Python3.9的电商销售数据自动化分析工具，支持数据生成、处理、可视化及PDF报告输出。

## 功能特性

### 1. 数据字段
- 订单ID、用户ID
- 下单时间（精确到分钟）
- 商品品类（8种）：手机数码、家用电器、服装鞋包、食品生鲜、美妆个护、家居家装、母婴用品、运动户外
- 单价、数量、总价

### 2. 可视化分析
- **每日销售柱状图**：按天展示销售总金额，附带趋势线
- **品类销售饼图**：展示销售占比，前4名品类单独展示，其余归为"其他"

### 3. PDF报告输出
- 两页精美报告，包含图表和详细说明文字
- 第1页：数据概览 + 每日销售趋势分析
- 第2页：商品品类结构分析 + 总结建议
- 支持重复运行，直接覆盖原有PDF

## 技术栈

- **Python**: 3.9+
- **数据处理**: Polars（高性能DataFrame库）
- **可视化**: Plotly（交互式图表）
- **PDF生成**: ReportLab

## 安装与运行

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行程序
```bash
python main.py
```

程序将自动执行以下步骤：
1. 生成300条2026年5月的测试销售数据（CSV格式）
2. 数据符合真实业务规律：黄金时段（10-12点、14-16点、19-22点）订单密集
3. 进行数据聚合与统计分析
4. 生成可视化图表
5. 输出完整的PDF分析报告至 `output/sales_analysis_report.pdf`

## 模块说明

### data_generator.py
- 支持指定数据量、年份、月份
- 分时段订单权重（黄金时段高权重）
- 各品类价格区间配置
- 订单数量概率分布

### data_processor.py
- Polars高性能数据处理
- 每日销售金额聚合
- 商品品类销售聚合
- Top N品类与"其他"合并
- 基础统计指标计算

### visualizer.py
- Plotly交互式图表生成
- 柱状图+趋势线组合展示
- 环形饼图带百分比标签
- 支持图片导出为字节

### pdf_generator.py
- ReportLab专业PDF排版
- 中文支持（PingFang字体）
- 关键指标表格展示
- 详细分析说明文案
- 两页版式设计
