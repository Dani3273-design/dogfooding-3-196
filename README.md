# 电商销售数据分析程序

基于 Python 3.9 的电商销售数据分析工具，支持自动生成测试数据、数据分析和 PDF 报告输出。

## 功能特性

- **数据生成**：自动生成符合业务规律的测试数据（支持黄金时段、品类价格区间等）
- **数据分析**：使用 Polars 进行高效数据处理
- **可视化图表**：使用 Plotly 生成交互式图表
- **PDF 报告**：自动生成包含图表和分析说明的专业报告


## 使用方法

### 1. 生成测试数据并生成报告

```bash
python main.py --generate-data
```

此命令将：
1. 生成 300 条 2026 年 5 月的销售数据
2. 保存数据到 `data/sales_data_202605.csv`
3. 分析数据并生成 PDF 报告到 `output/sales_report.pdf`

### 2. 仅生成报告（使用已有数据）

```bash
python main.py --data-file data/sales_data_202605.csv
```

### 3. 自定义参数

```bash
python main.py --generate-data --year 2026 --month 5 --records 500 --output output/custom_report.pdf
```

### 命令行参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--generate-data` | 生成测试数据 | 否 |
| `--data-file` | 销售数据文件路径 | `data/sales_data_202605.csv` |
| `--output` | PDF 报告输出路径 | `output/sales_report.pdf` |
| `--year` | 数据年份 | 2026 |
| `--month` | 数据月份 | 5 |
| `--records` | 生成数据记录数 | 300 |

## 数据生成规则

程序生成的测试数据遵循以下业务规律：

1. **时间分布**：
   - 高峰时段（19:00-22:00）：订单概率最高
   - 黄金时段（10:00-14:00, 19:00-23:00）：订单概率较高
   - 普通时段（8:00-10:00, 14:00-19:00, 23:00-24:00）：订单概率一般
   - 低谷时段（0:00-8:00）：订单概率最低

2. **价格区间**：不同品类有合理的价格区间
   - 手机数码：¥500 - ¥8,000
   - 家用电器：¥200 - ¥5,000
   - 服装鞋包：¥50 - ¥2,000
   - 食品生鲜：¥10 - ¥500
   - 美妆个护：¥30 - ¥1,000
   - 家居家装：¥100 - ¥3,000
   - 母婴用品：¥20 - ¥800
   - 运动户外：¥50 - ¥1,500

3. **购买数量**：根据品类特性设置合理的数量分布

## PDF 报告内容

报告包含以下内容：

1. **数据概览**：核心业务指标汇总
2. **每日销售趋势分析**：柱状图展示每日销售变化
3. **商品品类销售占比分析**：饼图展示 TOP4 品类及"其他"
4. **总结与建议**：基于数据的分析结论和建议

## 注意事项

- PDF 报告生成时会自动覆盖同名文件
- 程序会自动创建 `data/` 和 `output/` 目录
- 首次运行需要安装 kaleido 以支持 Plotly 图表导出
